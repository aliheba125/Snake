> يجمع هذا الملف نتائج كل المراحل: تفكيك ثابت (jadx/apktool/Blutter) + هندسة عكسية أصلية (Ghidra) + تحليل ديناميكي (Qiling/Unicorn)، مع **تدرّج ثقة صريح** لكل استنتاج.

```
Snake Engine (com.snake) v2.2.6  — أداة تعديل ألعاب تُباع باشتراكات

├── libapp.so  (Flutter/Dart)  = الواجهة الأمامية / لوحة الإدارة

│

      • يولّد كودًا في ذاكرة RWX، يهوك دوال Java، يعيد تعيين UID/PID
```

## 3. حدود التحليل (بصراحة تامة — استجابةً للمراجعة)

### 3.1 لماذا لا محاكي أندرويد كامل؟
البيئة **بلا KVM ولا تسريع عتادي**، والمكتبة **ARM64 فقط** (لا نسخة x86). لذلك محاكي أندرويد كامل (بواجهة + Frida) **غير ممكن تقنيًا هنا** — وليس خيارًا أهملته.

### 3.2 لماذا توقّفت المحاكاة العميقة؟ (جدار مقاومة المحاكاة)
بنينا طبقة نداءات نظام حقيقية (mmap يخصّص فعليًا، mprotect ينجح، تسجيل المسارات...). النتيجة:
- بملء ذاكرة RWX بأصفار → المُنشئات **تتعطّل بعد ~94 تعليمة** (تقفز إلى كعب لم يُكتَب = تعليمة غير معرّفة).
- بملئها بـ`RET` → المُنشئات **تدخل حلقات لا نهائية** (dispatchers مُسطّحة تحتاج قيمًا مفكوكة من بيئة حقيقية لتخرج).
- **الاستنتاج:** هذا سلوك مقاومة مُصمَّم عمدًا؛ فكّ السلاسل مربوط بحالة لا تكتمل خارج جهاز أندرويد حقيقي.

### 3.3 ما لم نستطع استخراجه ساكنًا/بالمحاكاة (ويحتاج جهازًا حقيقيًا)
- أسماء دوال JNI الفعلية (تُسجَّل عبر `RegisterNatives` وقت التشغيل بأسماء مفكوكة).
- السلاسل المفكوكة داخل النواة الأصلية.
- الملفات الفعلية المُنشأة وحركة الشبكة الكاملة (تحدث بعد التهيئة المحميّة).

### 3.4 تصحيحات دقّة (نزولًا عند ملاحظات المراجعة)
- «25,804 نداء نظام» من الفكّ الساكن = **أثر خادع غير موثوق** — مرفوض.
- «OLLVM» = **تعتيم شبيه بـ OLLVM** (دليل قوي، لا جزم بأداة).
- «توليد الكود وقت التشغيل» = **مُثبَت الآن** (Ghidra + Qiling)، لم يعد استنتاجًا.

## 4. الطريق الوحيد لإكمال ما تبقّى: جهاز حقيقي + Frida
1. `frida -U -f com.snake` مع سكربت يهوك `RegisterNatives` → خريطة (اسم↔مؤشّر).
2. هوك `mmap/mprotect/dlopen/openat/connect` → رصد كل تحميل/تعديل/اتصال فعلي.
3. `blutter_frida.js` (مرفق) لتتبّع دوال Dart.
4. تفريغ ذاكرة RWX بعد التهيئة → التقاط الكود بعد فك تعديله ذاتيًا.

> ملاحظة: بدون روت، الحقن يتطلّب إعادة تغليف بـ frida-gadget، وقد يكتشفه `chl` (فحص التوقيع). جهاز/محاكي مروّت هو الأنسب.
